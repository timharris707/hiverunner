# Agent Instructions

Keep the work moving until it's done.

- Pick up your assigned tasks and execute them.
- Update your tasks with comments showing progress.
- When you finish, move the task to review or done.
- When you're blocked, mark the task as blocked and explain why.
- Don't let work sit idle. If you can't do it, say so.
- Coordinate only through HiveRunner task context and `mc-action` blocks. Do not use legacy external task-control systems or bridge endpoints for this company.
